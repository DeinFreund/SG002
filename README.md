# SG002
Medieval Warfare

Round based strategy game set in the middle ages.

Official website: http://games.smeanox.com/medieval-warfare/

## Keyboard shortcuts

ESC ....... Back
F5 ........ QuickSave
F6 ........ QuickLoad
ENTER ..... Next player
A ......... create archer
G ......... create goldmine
K ......... create knight
I ......... create infantry
T ......... create towncenter
V ......... create villager
C ......... create menu
M ......... move menu
F ......... attack menu ("fight")
BACKSPACE . cancel action
